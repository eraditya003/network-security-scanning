# 🔐 Network Security & Scanning Project

## 📌 Objective
Perform network scanning, vulnerability analysis, and packet inspection using Kali Linux and Metasploitable.

---

## 🛠 Tools Used
- Kali Linux
- Metasploitable 2
- Nmap
- Wireshark
- Metasploit

---

## 🌐 Network Setup
- Kali IP: 192.168.56.101
- Target IP: 192.168.56.102

---

## 🔍 Steps Performed

### 1. Reconnaissance
- Ping test successful

### 2. Scanning
- Used Nmap to identify open ports and services

### 3. Vulnerability Analysis
- FTP vulnerable (vsftpd 2.3.4)
- Telnet insecure
- SMB vulnerable

### 4. Packet Analysis
- Captured ARP, DHCP, ICMP using Wireshark

### 5. Firewall
- Blocked port 80 using iptables

---

## 📊 Results
- Multiple open ports detected
- System highly vulnerable
